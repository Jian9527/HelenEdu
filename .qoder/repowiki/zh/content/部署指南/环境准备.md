## 简介

HelenEdu是一个基于Spring Boot和Vue.js的轻量级作业管理系统，专为海伦云作业场景设计。该项目采用前后端分离架构，后端使用Java技术栈，前端使用Vue.js框架，支持微信小程序和Web应用两种部署方式。

## 项目概述

HelenEdu项目包含两个主要组件：
- **后端服务**：基于Spring Boot 3.2.5，使用Java 17开发，提供RESTful API接口
- **前端应用**：基于Vue.js 3.4.21和UniApp框架，支持多端部署

项目采用现代化的技术栈，具有良好的扩展性和维护性。

## 硬件要求

### 最低配置要求
- **CPU**：Intel Core i3或同等AMD处理器
- **内存**：4GB RAM（建议8GB以上）
- **存储空间**：至少20GB可用磁盘空间
- **网络**：稳定的互联网连接

### 推荐配置要求
- **CPU**：Intel Core i5或更高性能处理器
- **内存**：8GB RAM（16GB以上最佳）
- **存储空间**：50GB以上可用磁盘空间
- **网络**：千兆网络连接

### 数据库专用配置
- **MySQL**：至少需要2GB内存分配给MySQL实例
- **缓存**：建议使用Redis作为缓存层（可选）

## 操作系统要求

### Windows系统
- **Windows 10/11**（推荐）
- **Windows Server 2016/2019/2022**
- **最低支持版本**：Windows 7 SP1

### Linux系统
- **Ubuntu**：20.04 LTS及以上版本
- **CentOS/RHEL**：8.0及以上版本
- **Debian**：10及以上版本
- **openSUSE**：15.3及以上版本

### macOS系统
- **macOS**：10.15及以上版本
- **推荐版本**：11.0及以上

## 网络配置要求

### 端口规划
- **后端服务端口**：8080（可在application.yml中修改）
- **数据库端口**：3306（MySQL默认端口）
- **开发调试端口**：3000（前端开发服务器）
- **Swagger文档端口**：8080（内置Swagger UI）

### 网络协议
- **HTTP/HTTPS**：支持标准HTTP协议
- **WebSocket**：支持实时通信功能
- **CORS配置**：已内置跨域资源共享支持

### 防火墙规则
- 允许8080端口的入站连接
- 允许3306端口的数据库连接
- 允许必要的出站连接用于包管理器

## 必需软件安装与配置

### JDK 17+ 安装

#### Windows安装步骤
1. 访问Oracle官网下载JDK 17+
2. 运行安装程序，选择默认安装路径
3. 完成安装后重启系统

#### Linux安装步骤
```bash
# Ubuntu/Debian系统
sudo apt update
sudo apt install openjdk-17-jdk

# CentOS/RHEL系统
sudo yum install java-17-openjdk-devel

# 验证安装
java -version
javac -version
```

#### macOS安装步骤
```bash
# 使用Homebrew安装
brew install openjdk@17

# 或者从Oracle官网下载安装
```

### MySQL 8.0+ 安装

#### Windows安装步骤
1. 下载MySQL Installer
2. 选择"Server only"安装模式
3. 设置root用户密码
4. 配置MySQL服务

#### Linux安装步骤
```bash
# Ubuntu/Debian
sudo apt install mysql-server

# CentOS/RHEL
sudo yum install mysql-server

# 启动MySQL服务
sudo systemctl start mysqld
sudo systemctl enable mysqld
```

#### 初始配置
```sql
-- 登录MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE helen_edu DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 创建用户并授权
CREATE USER 'helen'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON helen_edu.* TO 'helen'@'localhost';
FLUSH PRIVILEGES;
```

## 环境变量设置

### Java环境变量配置

#### Windows系统
```cmd
# 设置JAVA_HOME
setx JAVA_HOME "C:\Program Files\Java\jdk-17"

# 添加到PATH
setx PATH "%PATH%;%JAVA_HOME%\bin"

# 验证配置
java -version
javac -version
```

#### Linux/macOS系统
```bash
# 编辑~/.bashrc或~/.zshrc
export JAVA_HOME=/usr/lib/jvm/java-17-openjdk
export PATH=$PATH:$JAVA_HOME/bin

# 重新加载配置
source ~/.bashrc
```

# 设置MYSQL_HOME

## 防火墙配置

### Windows防火墙配置
1. 打开"Windows Defender防火墙"
2. 点击"高级设置"
3. 创建入站规则允许8080端口
4. 创建入站规则允许3306端口

### Linux防火墙配置（iptables）
```bash
# 允许8080端口
sudo iptables -A INPUT -p tcp --dport 8080 -j ACCEPT

# 允许3306端口
sudo iptables -A INPUT -p tcp --dport 3306 -j ACCEPT

# 保存规则
sudo iptables-save > /etc/iptables/rules.v4
```

### Linux防火墙配置（firewalld）
```bash
# 允许8080端口
sudo firewall-cmd --permanent --add-port=8080/tcp
sudo firewall-cmd --reload

# 允许3306端口
sudo firewall-cmd --permanent --add-port=3306/tcp
sudo firewall-cmd --reload
```

### macOS防火墙配置
1. 打开"系统偏好设置"
2. 点击"安全性与隐私"
3. 点击"防火墙选项"
4. 允许应用程序通过防火墙

## 常见问题排查

### Java相关问题

#### JDK版本不兼容
**问题症状**：编译错误或运行时异常
**解决方法**：
1. 检查当前JDK版本
2. 升级到JDK 17或更高版本
3. 更新JAVA_HOME环境变量

#### 内存不足问题
**问题症状**：编译失败或运行缓慢
**解决方法**：
```bash
# 增加JVM堆内存
export MAVEN_OPTS="-Xmx2048m -XX:MaxPermSize=512m"
```

### MySQL相关问题

#### 数据库连接失败
**问题症状**：应用启动时报数据库连接错误
**解决方法**：
```sql
-- 检查MySQL服务状态
sudo systemctl status mysqld

-- 验证用户权限
SELECT User, Host FROM mysql.user;

-- 重置root密码
ALTER USER 'root'@'localhost' IDENTIFIED BY 'new_password';
```

#### 字符集编码问题
**问题症状**：中文显示乱码
**解决方法**：
```sql
-- 修改数据库字符集
ALTER DATABASE helen_edu CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 修改表字符集
ALTER TABLE sys_user CONVERT TO CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 网络连接问题

#### 端口被占用
**问题症状**：服务无法启动
**解决方法**：
```bash
# 查找占用端口的进程
lsof -i :8080

# 终止进程或更改端口
kill -9 PID
```

#### CORS跨域问题
**问题症状**：前端请求被阻止
**解决方法**：
检查application.yml中的CORS配置，确保允许的源地址正确设置。

### 前端构建问题

#### 依赖安装失败
**问题症状**：npm install报错
**解决方法**：
```bash
# 清理缓存
npm cache clean --force

# 删除node_modules重新安装
rm -rf node_modules
npm install

# 使用淘宝镜像
npm install --registry https://registry.npmmirror.com/
```

#### 热更新失效
**问题症状**：代码修改后页面不刷新
**解决方法**：
```bash
# 检查Vite配置
cat vite.config.js

# 重启开发服务器
npm run dev:h5
```

### 项目启动问题

#### 应用启动失败
**问题症状**：Spring Boot应用无法启动
**解决方法**：
```bash
# 检查日志输出
tail -f target/classes/application.yml

# 验证配置文件语法
java -jar helenedu-backend/target/helenedu-backend-1.0.0.jar --spring.profiles.active=dev
```

#### 依赖冲突问题
**问题症状**：Maven构建失败
**解决方法**：
```bash
# 清理并重新构建
mvn clean compile

# 检查依赖树
mvn dependency:tree

# 更新依赖版本
mvn versions:display-dependency-updates
```
